Privacy Policy for HTML Viewer Web App

Effective date: December 25, 2025

1. About the App

HTML Viewer is a web app that allows users to input and preview HTML code directly in their browser. The app is simple, lightweight, and does not require any login or registration.

2. Information Collection

HTML Viewer Web App does not collect, store, or share any personal or sensitive user data. All HTML code processing happens locally within the user’s browser.

3. Data Usage

No user data is collected, transmitted, or stored. The app works entirely on the user’s device without any backend server involvement.

4. Third-Party Services

The app does not use or integrate with any third-party services, SDKs, analytics tools, or advertising networks.

5. Security

Since the app does not collect or store any data, there is no risk of personal information exposure. All code entered by users remains private within their browser session.

6. Children’s Privacy

This app is suitable for all age groups. We do not knowingly collect any information from children under the age of 13.
